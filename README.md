## Description

> Netflix를 참고하여 제작하였습니다.

> Movie 와 TV 시리즈에 대한 소개와 제작 및 정보를 보여주는 Netflix 클론 웹사이트 입니다. (영상 재생은 생략)

> "The Movie Database" 에서 무료로 제공하는 api를 통해 Movie, TV 데이터를 사용하였습니다.

> 링크 : 오른쪽 "About" 참조
